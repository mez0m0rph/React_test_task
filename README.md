## Что сделал

В приложении можно:

- отобразить дерево с вложенными узлами
- выбрать узел
- добавить узел-потомок к выбранному элементу
- изменить имя выбранного узла
- удалить выбранный узел
- сбросить дерево к начальному состоянию

## Структура 

- `src/components` — React-компоненты (интерфейс)
- `src/data` — исходные данные дерева
- `src/utils` — доп функции для работы с деревом (со структурой)

## Идея решения (как делал)

Дерево хранится в состоянии корневого компонента.

У узла такая структура:

- `id`
- `name`
- `children`

Отрисовывается дерево рекурсивно через `TreeNode`.

Добавление, удаление, редакт - вынес в helper-функции. Они возвращают обновленную версию дерева, сохраняя immutable.
